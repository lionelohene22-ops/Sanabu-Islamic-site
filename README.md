# Sanabu Islamic Site

A modern, responsive website for Sanabu Islamic community.

## Website URL
**https://sanabuislamic.com**

## Features

- **Home Page**: Welcome and introduction to Sanabu Islamic
- **About Page**: Information about our mission and services
- **Contact Page**: Contact form and information
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Clean Styling**: Professional and user-friendly interface

## Pages

1. `index.html` - Home page
2. `about.html` - About us page
3. `contact.html` - Contact page
4. `styles.css` - Styling for all pages

## Getting Started

1. Clone the repository
2. Open any HTML file in your browser
3. Navigate using the menu links

## Customization

You can customize:
- Content in each HTML file
- Colors in `styles.css` (currently using green theme: #1a472a)
- Add more pages as needed
